      • It is almost similar to RGB format except that RGBA contains A (Alpha) that specifies the element's transparency. The value of alpha is in the range 0 to 1, in which 0 is for fully transparent, and 1 is for not transparent.

      • Hexadecimal can be defined as a six-digit color representation. This notation starts with the # symbol followed by six characters ranges from 0 to F. In hexadecimal notation, the first two digits represent the red (RR) color value, the next two digits represent the green (GG) color value, and the last two digits represent the blue (BB) color value.
      Syntax:color:#(0-F)(0-F)(0-F)(0-F)(0-F)(0-F);

      • It is a short form of Hue, Saturation, and Lightness. Hue: It can be defined as the degree on the color wheel from 0 to 360. 0 represents red, 120 represents green, 240 represents blue. Saturation: It takes value in percentage in which 100% represents fully saturated. Lightness: The lightness of the color can be defined as the light that we want to provide the color in which 0% represents black (there is no light), 50% represents neither dark nor light, and 100% represents white (full lightness).
      Syntax:color:hsl(H, S, L);
    • HSLA:
      • It is similar to HSL property, except that it contains A (alpha) that specifies the element's transparency.
      Syntax: color:hsla(H, S, L, A);
    • Built-in color:
      • As its name implies, built-in color means the collection of previously defined colors that are used by using a name such as red, blue, green, etc.

    Q:12 How to maintain the CSS specifications?


    Ans.

  • If two rules are applied to the same element, the one that comes last is the one that will be used, and its style will be applied to that HTML element.

  • If two CSS selectors apply to the same element, the one with higher specificity is used.
    • For Example:

    Ex.1 We'll apply two classes to an element and give each class a different background-color.

    HTML Code:

    <p class="style1 style2"> This is a test paragraph</p>

    CSS Code:

    .style2{
        background-color: red;
        }
    .style1{
       background-color: yellow;
       }

    This is a test paragraph

    The style1 which comes last in the stylesheet is applied to the element.

    Ex.2We'll apply one Class and one Id to an element and give a different background-color.

    HTML Code:

    <p class="example2" id="paragraph"> This is a test paragraph</p>

    CSS Code:

    #paragraph{
       background-color: red;
       }
    .example2{
        background-color: yellow;
       }

    This is a test paragraph

    The #paragraph is an id selector, while the .example2 is a class selector.The property value of the id selector was applied because it has a higher specificity value.

    Ex.3 if we have an element and for the same property color, there's an inline style.

    HTML Code:

    <p style="color: red" class="yellow" id="paragraph">This is a paragraph</p> CSS Code:

    #paragraph1{
        color: green;
        }
    .yellow{
           color: yellow;
       }

    This is a paragraph

    If the class and id selectors also have values for the same property, the inline style wins because it has a higher specificity value.

    Note: The weights only apply when different selectors have values for the same property.

    Ex.4 if we have an element and for the same property color, there's an inline style.

  • Applying !important to the property value of any selector makes it the value that will be applied to the element.

    • HTML Code:

    <p class="blue" id="paragraph" style="color: green"> This is a paragraph </p> CSS Code:

    p{
      color: red !important;
     }
    .blue{
       color: blue;
       }
    #paragraph{
        color: purple;
       }

    This is a paragraph

    That's why !important should be avoided because it makes it difficult override a style.

    Q:16 What are the advantages and disadvantages of using external style sheets?


    Ans.

    Introduction:

       Apply CSS in an external file and link it using link tag.We can linked or imported CSS to External CSS.

    Advantages of external style sheet:

    1. Using External Style Sheets, the styles of various documents can be organized in one single file.
    2. An external style sheet can be reused in multiple files, saving time and making it easier to maintain a consistent look.
    3. An external style sheet allows you to separate the content of your HTML from the CSS. This makes it easier to maintain and update your HTML, as you only need to change the style sheet rather than making changes to the HTML of each individual file.

    Disadvantages of external style sheet:

    1. An extra download is essential to import style information for each file.
    2. Importance of Internal CSS and Inline CSS is heigher than External CSS.

    Q:17 What is the meaning of the CSS selector?


    Ans.


    Meaning:

  • CSS selectors are used to select the content you want to style. CSS selectors select HTML elements according to its id, class, type, attribute etc.

    • Types of Selector:
    • Element selector:
      • Ex. p{} here, p is element selector
    • Class selector:
      • Ex. .text{} here, text class is class selector (For class we use (.))
    • Id selector:
      • Ex. #text{} here, Id text is Id selector (For Id we use (#))
    • Universal selector:
      • Ex. *{} here, * referce every element used in body (For universal selector we use (*))
    • Adjecent sibling selector:
      • Ex. h1+p{} here, If we give css to h1 tag and use adjecent sibling selector then 1st p tag get diffrent css than ther p tag (For this selector we use(+))
    • General sibling selector:
      • Ex. h1~p{} here, If we want to give give css to p tags which is after h1 tag than we can use general sibling selector (For this selector we use (~))
    • Group selector:
      • Ex. h1,p,b{} here, If we want give same CSS to more than one element we can use group selector.
    • Child selector:
      • Ex. div > img{} ,If we want to give css to an element which is inside of another element than we can use child selector (For this selector we use (>) or space between both element like(div img{}) )
