How to embed audio and video in a webpage?



Ans:

Audio:

To embed audio in webpage, we use the < audio> tag.

After the release of HTML5, it is possible to play the audio in browser.

Audio Tag supports 3 audio formats: mp3, ogg, WAV.

Syntax:

< audio>
< source src="file_name" type="file_type">
< /audio>

Example:



Video:

To embed video in webpage, we use the < video> tag.

After the release of HTML5,it is possible to play one or more video sources at a time using < source> tag.

Video Tag supports 3 video formats: mp4, WebM, and ogg.

Syntax:

< video>
< source src="file_name" type="file_type">
< /video>

Example:


* Audio and Video tag are many attribute.

Attribute Description
autoplay When the page is loaded. It specifies to play audio/video as soon as possible.
controls It displays audio/video control such as play, pause, and stop.
loop Used to start the audio/video again when it is finished.
muted When the page is loaded audio/video will be automatically muted.
src It specifies the URL of the audio/video file.
