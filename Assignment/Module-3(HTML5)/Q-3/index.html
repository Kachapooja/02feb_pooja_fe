Semantic element in HTML5?



Ans:

Semantic element which means types of content.

Semantic elements are header, footer, nav, etc....

  • < article>
  • < header>
  • < aside>
  • < main>
  • < details>
  • < mark>
  • < figure>
  • < nav>
  • < figcaption>
  • < section>
  • < footer>
  • < article>

    • defines as a blog entry or newspaper article

    Example:

    Google Chrome

    Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!

  • < aside>

    • defines as a some content other than the content placed in it.

    Example:

    Epcot Center

    Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.

    • used to the added additional details.

  • < footer>

    Mark Tag!
