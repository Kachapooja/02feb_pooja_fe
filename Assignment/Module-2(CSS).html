What is the difference between CSS2 and CSS3?



Ans:

Difference between CSS2 & CSS3

CSS2 CSS3
1. There are new ways you can write CSS rules with a bunch of CSS selectors. 1. There were no new ways of writing the CSS rules.
2. There is no backward compatibility with CSS2 2. There is backward compatibility maintained with CSS3
3. In CSS2 only web safe fonts can be used 3. In CSS3 special fonts can be used such as those in Google fonts.
4. In CSS2 the concept of simple selectors were present 4. In CSS3 the selectors were called as a sequence of simple selector
5. Using CSS2,for rounded borders,coding the css styles were complex 5. Using CSS3,there is provision for automatically assigning rounded borders to objects21
6. CSS2 doesn't support the Border-Box property 6. CSS3 supports the Border-Box property
7. CSS2,splitting text into multiple columns required complex coding. 7. CSS3 has the capability to split text into various columns.

What are the ways to integrate CSS as a web page?



Ans:

CSS can be added to HTML documents in 3 ways:
  1. Internal
  2. Inline
  3. External
1.Internal:

  • An Internal CSS is used to define a style for a single HTML page.

  • An Internal CSS is using a <style> element in the <head> section.

Example:

<head>
<style>
.internal {
   background-color: yellogreen;
   color: #000;
   font-size: 30px;
   padding: 5px;
}
</style>
</head>

Output:

Internal CSS


2.Inline:

  • An Inline CSS is used to apply a unique style to a single HTML element.

  • An Inline CSS uses the <style> attribute of an HTML element.

Example:

< p class="inline" style="background-color: powderblue; padding: 5px; font-size: 30px;"> Inline CSS < /p>

Output:

Inline CSS


3.External:

  • An external CSS is used to define the style for many HTML pages.

  • An external CSS add a link to it in the < head> section of each HTML page.

Example:

< head>
< link rel="stylesheet" href="Module-2(CSS).css">
< /head>
< body>
< p class="external"> External CSS < /p>
< /body>

External CSS

.external {
   background-color: tomato;
   color: #000;
   font-size: 30px;
   padding: 5px;
}

Output:

External CSS

Que:16

What are the advantages and disadvantages of using external style sheets?



Ans:

The advantages of External Style Sheets:
  • In External Stylesheet the styles of multiple documents can be controlled from one file.
  • In External Stylesheet classes can be created for use on multiple HTML element types in many documents.
  • In difficult situations, Selector and Grouping methods can be used to apply styles.
The disadvantages of External Style Sheets:
  • In order to import style information for each document, an extra download is needed.
  • Until the external style sheet is loaded, it may not be possible to submit the document.
  • For small number of style definitions, it is not possible.

What is the meaning of the CSS selector?



Ans:

CSS selectors are used to select the HTML elements we want to style. CSS selectors are devide in 6 categories:
  1. Element Selector
  2. Class Selectors
  3. Id Selectors
  4. Universal Selectors
  5. Adjucent Selectors
  6. General Selectors
  7. Group Selectors
  8. Child Selectors
    9.

Example:

<head>
<style>
  p {
    text-align: center;
    color: purple;
    padding: 2px;
    background-color: teal;
  }
</style>
</head>
< body>

   < p>Also Paragraph Tag!< /p>
< /body>

Every paragraph will be affected by the style.



Paragraph Tag!



Also Paragraph Tag!

Example:

<style>
  .colour {
    color: orangered;
    font-size: x-large;
  }
<style>
< body>

Example:

    background-color: lightblue;
  }

5.Adjucent Selector: Adjucent(immediately follow) Sibling Selectors(+) is used to select element that is directly after another specific element.

Example:

If we give css to div tag and use Adjecent Sibling Selector then 1st p tag get diffrent css than their p tag.

CSS

  div + p {
    background-color:crimson;
    color:#fff;
  }

6.General Selector: General Sibling Selectors(~) is used to selects all elements that are next siblings of a specified element.

Example:

If we give css to div tag and use General Sibling Selector then next line p tag get same css.

CSS

  div ~ p {
    background-color:crimson;
    color:#fff;
  }

8.Group Selector:The Group Selectors is used to all the HTML elements with the same style definitions.

CSS

  h1, h2, p {
    background-color:crimson;
    color:#fff;
  }

9.Child Selector:The Child Selector(>) selects is used an element which is inside of another element.

Example:

CSS

  div > p {
    background-color:crimson;
    color:#fff;
  }
