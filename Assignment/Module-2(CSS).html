What are the disadvantages of CSS?

Ans:

Disadvantages of CSS:
  1. Confusion due to many CSS levels

    2. Beginners might get confused while opting to learn CSS as there are many levels of CSS such as CSS2, CSS3....

  2. Cross-Browser Issues

    3. Different browsers work differently.We have to check that changes implemented in the website via CSS codes are reflected properly among all browsers.

  3. Security Issues

    4. Security is important in today's world driven by technology and data. One of the major disadvantages of CSS is that it has limited security.

  4. Extra Work for Developers

    5. Design services are required to consider and test all CSS codes across different browsers for compatibility. Due to developers testing compatibility for different browsers, their workload increases.

What is the difference between CSS2 and CSS3?

Ans:

Difference between CSS2 & CSS3

CSS2 CSS3
1. There are new ways you can write CSS rules with a bunch of CSS selectors. 1. There were no new ways of writing the CSS rules.
2. There is no backward compatibility with CSS2 2. There is backward compatibility maintained with CSS3
3. In CSS2 only web safe fonts can be used 3. In CSS3 special fonts can be used such as those in Google fonts.
4. In CSS2 the concept of simple selectors were present 4. In CSS3 the selectors were called as a sequence of simple selector
5. Using CSS2,for rounded borders,coding the css styles were complex 5. Using CSS3,there is provision for automatically assigning rounded borders to objects21
6. CSS2 doesn't support the Border-Box property 6. CSS3 supports the Border-Box property
7. CSS2,splitting text into multiple columns required complex coding. 7. CSS3 has the capability to split text into various columns.

Name a few CSS style components

Ans:

  1. Selector:

    2. "Class" name, "Id" name or "Element" name that is target.

  2. Attribute:

    3. Name of the attribute you want to style for example border, color, background, position etc.

  3. Value of Property:

    4. Value that will be assigned to attribute.

Example:

    p{
     color: red;
     position: relative;
     text-align:left;
    }
  • p is a selector and value is red.
  • position is attribute and value is relative.
  • text-align is attribute and value is left.
