Que:1

Are the HTML tags and elements the same thing?

Ans:

HTML tags and elements are not same thing.
  • HTML Tags are the starting and ending parts of an HTML element. They begin with "<" symbol and end with ">" symbol. Whatever written inside < and > are called tags.

    • Example: < b>......< /b>

  • Some HTML tags do not have end tags like < img >.

  • HTML element consists of the start tag + content + end tag.

    • Example: < b>This is the content.< /b>

What are tags and attributes in HTML?

Ans:

HTML Tags
  • HTML Tags are used to mark up the start of an HTML element and they are usually enclosed in angle brackets.

  • Most tags must be opened < h1> and closed < /h1> in order to function.
Example: < h1>, < HTML>, < ! Doctype html>, < head>, < span>, < body>, < p>, < div>, < u>, < sub>, < sup>, < strong>, < li>, < ol>, < ul>.

HTML Attributes
  • Attributes provide additional information about elements

  • Attributes are always specified in the start tag and it usually come in name/value pair:-name="value"
Example: < a href="https://www.Google.com">Google< /a>

Que:3

What are void elements in HTML?

Ans:

  • There is a special group of elements that only have start tags and does not contain any content within it, these elements are called Void elements.
  • Void elements doesn't have ending tags and can only have attributes.
  • Void elements cannot have content inside it.
Example: < br>, < hr>, < img>, < input>, < link>, < base>, < meta>, < area>, < embed>, < col>

What are HTML Entities?

Ans:

  • HTML Entities are basically a set of Entities used to represent few characters reserved by the HTML, especially invisible characters or characters difficult to type out using a regular keyboard.
  • HTML entities is a text ("string") that begins with an (&) and ends with a (;).
Example: 1. < - & lt; 2. > - & gt;
