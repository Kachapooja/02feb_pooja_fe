How to center block elements using CSS1?



Ans:

There are two steps to center a block-level Element:-

  1. Width

    2. We need to define the external width.

    Block-level elements have the default width of 100% of the webpage.

    so for centering the block element,we are giving to the external width.

  2. Margin

    3. we need to align that space properly that's why we should use margin property.

    So for centering the element we must set left-margin to auto and right-margin to auto.
