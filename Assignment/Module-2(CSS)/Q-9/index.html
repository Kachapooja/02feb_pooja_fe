Que:9

Which property controls the image scroll in the background?



Ans:

  • The Background-attachment property in CSS is used to specify the kind of attachment of the background image.
  • The Background-attachment property set to scroll or make it remain fixed.
  • The Background-attachment property applied to all HTML elements.
1.Background-attachment: Scroll
