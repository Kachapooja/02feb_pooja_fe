Que:8

What is the use of the background-position property?



Ans:

Background-position property set to the starting position of a background image.

The Background-image is placed default to the top-left corner of an element with a repetition on both horizontally & vertically.

Background-position
  • background-position: left top
  • background-position: left center
  • background-position: left bottom
  • background-position: center top
  • background-position: center center
  • background-position: center bottom
  • background-position: right top
  • background-position: right center
  • background-position: right bottom
  • background-position: 25% 75%
  • background-position: 30px 80px
Example:

Background-position: left top

CSS

.background-position{
    background-image: url("https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTx-EqWsFXuHleQfVIuV5kvVXkEznd7eG8869yAbUU59w&usqp=CAU&ec=48600112");
    background-repeat: no-repeat;
    background-position: left top;
    border: 1px solid black;
    background-color: brown;
    height: 300px;
    width: 350px;
}
