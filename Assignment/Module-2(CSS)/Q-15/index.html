What are the external style sheets?



Ans:

  • External style sheet is a separate CSS file that can be accessed by creating a link within the Head section of the webpage.

  • Multiple webpages can be use the same link to access the stylesheet.

  • It uses the < link> tag on every pages and the < link> tag should be put inside the Head section.

  • External style sheet may be written in any text editor but must be saved with a .css Extension.
Example:

< head>
< link rel="stylesheet" href="Module-2(CSS).css" type="text/css">
< /head>
< body>
< p class="sheet"> External Stylesheet < /p>
< /body>

External CSS

.sheet {
   background-color: lightseagreen;
   color: #000;
   font-size: 15px;
   padding: 5px;
   display:inline;
}
