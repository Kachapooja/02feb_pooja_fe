What is embedded style sheets?



Ans:

  • Embedded Stylesheet define styles for a particular HTML document as a whole in one place.

  • Embedded Stylesheet done by embedding the <style></style> tags containing the CSS properties in the Head of HTML document.

  • Embedded Stylesheets is exactly the same as other CSS code.

Example:

<head>
<style>
  h7 {
    color: #fff;
    background-color: teal;
    padding: 5px;
  }
  i{
    font-size:10px;
    background-color:crimson;
    Color:#fff;
    padding:5px;
    display:block;
  }
</style>
</head>
<body>
    < h7>Heading H7 Tag!< /h7>
    < i> Italic Tag!< /i>
</body>

Output:

Heading H7 Tag!

Italic Tag!
