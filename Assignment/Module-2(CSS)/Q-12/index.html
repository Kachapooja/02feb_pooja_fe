Que:12

How to maintain the CSS specifications?



Ans:

When more than one CSS rule is applied to an element, Browser decides which specific element to apply to.The group of rules that bronzers follow is called a specification.

Example:-1

we will apply to CSS style by external stylesheet it has become to the lowest precedence compare to internal and inline CSS.

CSS:

<head>
<style>
  strong {
    background-color: red;
    color: white;
   }
</style>
</head>

HTML:

   < body>
    < strong>Strong Tag!< /strong>
   < /body>

Output:

Strong Tag!

Example:-2

we will apply to class selector and one element. Apply to crimson color for class and green color for element.Now text will show the green color because class is given higher priority.

CSS:

<head>
<style>
  .test {
    color: crimson;}
    cite{color: green;
   }
</style>
</head>

HTML:

   < body>
    < cite class="test">Cite Tag!< /cite>
   < /body>

Output:

Cite Tag!

Example:-3

we will apply to id selector and class selector.Apply to marron color for id and orange color for the class.Now text will show the maroon color because id is given higher priority.

CSS:

<head>
<style>
  #test-1{
    color:maroon;
   }
  .test-2 {
    color: orange;}
    i{color: green;
   }
</style>
</head>

HTML:

   < body>
    < i class="test">italic Tag!< /i>
   < /body>

Output:

italic Tag!

Example:-4

we will apply to inline style for the "em" element.Apply to color in inline style to the pink.Now the inline style is given higher priority.

CSS:

<head>
<style>
  #test-3{
    color:blue;
   }
  .test-4 {
    color: orangered;}
    em{color: green;
   }
</style>
</head>

HTML:

   < body>
    < id="test-3" class="test-4" style="color: red;">Emphasized Tag!< /em>
   < /body>

Output:

Emphasized Tag!

Now the every element has own priority and position.

  • Inline: Inline style has higher priority.

  • ID: ID have the 2nd priority.

  • Class: Classes,pseudo-class and attribute are next priority.

  • Element: Elements,pseudo-elements have lower priority.
