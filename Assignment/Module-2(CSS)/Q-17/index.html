What is the meaning of the CSS selector?



Ans:

CSS selectors are used to select the HTML elements we want to style. CSS selectors are devide in 6 categories:
  1. Element Selector
  2. Class Selectors
  3. Id Selectors
  4. Universal Selectors
  5. Adjucent Selectors
  6. General Selectors
  7. Group Selectors
  8. Child Selectors
    9.
1.Element Selector: The Element Selector selects HTML elements based on the element name.

Example:

<head>
<style>
  p {
    text-align: center;
    color: purple;
    padding: 2px;
    background-color: teal;
  }
</style>
</head>
< body>
   < p>Every paragraph will be affected by the style.< /p>
   < p id="p1">Paragraph Tag!< /p>
   < p>Also Paragraph Tag!< /p>
< /body>

Output:

Every paragraph will be affected by the style.



Paragraph Tag!



Also Paragraph Tag!



2.Class Selector: The Class Selector(.) selects HTML elements with a specific class attribute.

Example:

<style>
  .colour {
    color: orangered;
    font-size: x-large;
  }
<style>
< body>
    < div class="colour">Div Class Tag!< /div>
    < item class="colour">Item Class Tag!< /p>
< /body>

Output:


Div Class Tag!
Item Class Tag!

3.Id Selector: The Id element (#) is unique within a page, so the id selector is used to select one unique element!

Example:

<style>
  #id1 {
    color: green;
    font-size: x-large;
    background-color: lightseagreen;
  }
<style>
< body>
    < div id="id1">Id Selectors
    < div>Div Tag!
< /body>

Output:


Id Selectors


Div Tag!

4.Universal Selector: The universal selector (*) selects all HTML elements on the page.

Example:

<style>
  * {
    color: navy;
    font-size: x-large;
    background-color: lightblue;
  }
<style>
< body>
< u>Underline Tag!< /u>
< mark>Mark Tag!< /mark>
< del>Delete Tag!< /del>
< ins>Insight Tag!< /ins>
< /body>

Output:

Underline Tag!
Mark Tag!
Delete Tag!
Insight Tag!

5.Adjucent Selector: Adjucent(immediately follow) Sibling Selectors(+) is used to select element that is directly after another specific element.

Example:

If we give css to div tag and use Adjecent Sibling Selector then 1st p tag get diffrent css than their p tag.

CSS

  div + p {
    background-color:crimson;
    color:#fff;
  }

6.General Selector: General Sibling Selectors(~) is used to selects all elements that are next siblings of a specified element.

Example:

If we give css to div tag and use General Sibling Selector then next line p tag get same css.

CSS

  div ~ p {
    background-color:crimson;
    color:#fff;
  }

8.Group Selector:The Group Selectors is used to all the HTML elements with the same style definitions.

CSS

  h1, h2, p {
    background-color:crimson;
    color:#fff;
  }

9.Child Selector:The Child Selector(>) selects is used an element which is inside of another element.

Example:

CSS

  div > p {
    background-color:crimson;
    color:#fff;
  }
