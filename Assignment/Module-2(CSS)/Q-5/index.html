What do you understand by CSS Opacity?



Ans:

The CSS Opacity property is used to specify the transparency of an element.

In short, Opacity specifies the clarity of the image.

We can apply the opacity with different styling properties to the elements.

Example:

< div class="image">
< img src="https://cdn.pixabay.com/photo/2014/02/27/16/10/flowers-276014__340.jpg" alt="flowers">
< /div>
