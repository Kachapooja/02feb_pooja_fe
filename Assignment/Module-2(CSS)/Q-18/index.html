What are the media types allowed by CSS?



Ans:

The @media rules presented in CSS make it possible to define different style types for different media types.

@media can make one set of style rules for computer screens, one for printers, one for mobile devices, one for television-type devices, etc.

Media queries can be used to check many things.

  • width and height of the viewport
  • width and height of the device
  • orientation
  • resolution
Using media queries we can make style sheet to desktop,laptop,tablet and mobile screen.

Syntax:

@media mediatype and (expressions) {
     CSS-Code;
}
In media query we will use the diffrent style sheet by css.

all
Used for all media type devices
print
used for printers
screen
used for computer screen,tablets,mobile phone,etc..
speech
used for screenreaders that "reads" the page out loud
