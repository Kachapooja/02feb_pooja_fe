What is the rule set?



Ans:

  • A CSS ruleset is various support to various pieces or elements of the document.

  • A CSS Ruleset is the main building block of the CSS stylesheet.

  • A CSS ruleset will be applied when the selector name is called on the main HTML page.

  • A CSS “.” in the beginning indicates that the rule created will be a class, also “container” indicates the name of the selector.

  • In CSS also “first-child” indicates the pseudo-class, and elements inside "{}" are elements of a declaration block, which contains some CSS properties and their corresponding values.

Example:

<style>
  h4{
    color: navy;
  }
  ins:first-child {
    color: sienna;
    background-color: aqua;
    font-size: 20px;
    border-radius: 15%;
  }
</style>
< div>
   < ins>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quisquam, quo?
   < h4>This is a Heading H4 Tag!
   < h6>This is a Heading H6 Tag!
   < ins>This is a also insight!
< /div>
